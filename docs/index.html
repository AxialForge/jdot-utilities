Runs offline

No server, no localhost, no upload step. Install the app and convert — airplane mode works fine.

No account

No sign-up, no JWT, no history database. Open it and go straight to work.

Built to extend

Every tool is one self-describing file. Drop it in and the app renders the interface for it automatically.

Adding a tool is one file

The interface is generated from the tool registry — the tool rail, output formats, and option fields all come from what a tool exports. There's nothing else to wire up.

// src/tools/data-convert.js
module.exports = {
  id: "data-convert",
  name: "Data Converter",
  category: "Data",
  inputFormats: ["json", "yaml", "csv"],
  outputFormats: ["json", "yaml", "csv"],
  async convert({ inputPath, outputPath, outputFormat }) {
    /* read, transform, write — pure JS or a bundled binary */
  },
};
